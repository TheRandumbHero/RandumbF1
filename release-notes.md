# RandumbF1 v0.1.0 Beta

## Overview

This is the first public beta release of RandumbF1.

RandumbF1 is a standalone Formula 1 dashboard for CYD, providing race information, race calendar, driver standings, constructor standings, track overviews, track records and circuit information directly from the OpenF1 API.

Although feature complete enough for public testing, this is still a beta release and additional functionality, visual improvements and optimisations are planned before the first stable release.

---

## Features

### Race Information

- Displays the next Formula 1 race weekend
- Circuit map for the selected race
- Race weekend schedule
- Circuit information page
- Lap record, driver and year
- Full-screen circuit maps
- Swipe between circuit map and information pages

### Championship Standings

- Current Driver Championship standings
- Driver
- Constructor
- Championship points

### Race Calendar

- Browse the full Formula 1 calendar
- Select any future race
- View race details
- View circuit information
- View weekend session schedule

### Device Settings

- WiFi configuration
- Timezone selection
- Display colour inversion
- Team livery themes
- Factory reset

### Web Dashboard

- Responsive web interface
- WiFi setup
- Device configuration
- Theme selection
- Automatic reboot after saving settings

## Known Limitations

- Circuit maps are currently placeholders and will be replaced with improved artwork in a future release.
- Team and driver information will continue to be expanded.
- Additional race statistics are planned.
- Constructor standings are not yet available.
- Live timing won't be implemented as this is a paid requirement of OpenF1 and i'm trying to keep this as cheap as possible.
