Installation:
1. Connect the RandumbF1 device by USB.
2. Click Install below.
3. Select the correct serial port.
4. Follow the browser prompts.
5. Wait for the device to restart.
This browser does not support Web Serial. Use Chrome or Edge on a desktop computer.

The initial installation erases the device and installs RandumbF1 using the correct partition layout.
